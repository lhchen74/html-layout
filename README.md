### 两列布局

两列布局一般情况下是指定宽度与子适应布局，两列中左列是确定宽度，右列是自动填满剩余空间的一种效果。

实现方式：

- float + margin
- float + overflow
- display table

### 三列布局

定宽 + 定宽 + 自适应
定宽 + 自适应 + 定宽（圣杯布局)
优化圣杯布局需要开启定位的问题（双飞翼布局）

### 等高布局

等高布局就是一行被划分成若干列，每一列的高度都是相同值。

- display table
- padding + margin
